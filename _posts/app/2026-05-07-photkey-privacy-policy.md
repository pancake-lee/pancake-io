# Privacy Policy / 隐私政策

**English**

**Effective Date:** [May 7, 2026]

This privacy policy applies to the PhotKey application (hereinafter referred to as "this Application") developed by pancake.

**Information Collection and Use**

This Application does not collect, store, or transmit any personal information. All data, including configuration files and preferences, are stored locally on your device and are never uploaded to any server or shared with any third parties.

**Permissions**

This Application requires the `runFullTrust` permission to function properly. This permission is necessary to:

1. Register and monitor global system-level hotkeys.
2. Run keyboard input hooks for key interception and simulation.
3. Access and modify configuration files within the user directory.

**Changes to This Privacy Policy**

This Application may update its privacy policy from time to time. Users are advised to review this page periodically for any changes.

**Contact Us**

If you have any questions about this privacy policy, please contact me at: [pancake-lee@outlook.com](mailto:pancake-lee@outlook.com)

**中文**

**生效日期**: [2026年5月7日]

本隐私政策适用于 pancake 开发的 PhotKey 应用程序（以下简称“本应用”）。

**信息收集与使用**

本应用**不会收集、存储或传输任何个人信息**。所有数据，包括配置文件和偏好设置，都仅存储在您的本地设备上，不会上传至任何服务器，也不会与任何第三方共享。

**权限使用**

本应用需要 `runFullTrust` 权限才能正常运行。此权限是必要的，用于：

1. 注册并监听全局系统级热键。
2. 运行键盘输入挂钩，用于拦截和模拟按键。
3. 访问和修改用户目录下的配置文件。

**隐私政策的变更**

本应用可能会不时更新其隐私政策。建议用户定期查阅本页面以了解任何变更。
